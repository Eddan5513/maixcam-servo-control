# 🚀 Quick Start Guide

## Быстрая установка и запуск за 5 минут

### 1. Подготовка оборудования

#### Что нужно:
- MaixCAM (первое поколение с экраном)
- Сервопривод SG90 или аналогичный
- 3 провода для подключения
- Источник питания 5V (через USB или внешний)

#### Подключение сервопривода:
```
Servo Signal (желтый/белый) -> A18
Servo GND (коричневый/черный) -> GND
Servo Power (красный)        -> VBUS 5V
```

**⚠️ Важно:** Убедитесь что источник питания обеспечивает минимум 500mA для стабильной работы сервы.

### 2. Подключение к MaixCAM

#### Через WiFi:
1. Включите MaixCAM
2. Подключитесь к WiFi сети камеры или подключите камеру к вашей сети
3. Откройте MaixVision
4. Подключитесь к камере по IP: `10.144.30.1` (или IP в вашей сети)

#### Проверка подключения:
```python
# В MaixVision терминале
ping 10.144.30.1
```

### 3. Загрузка и установка (v2.0.0)

#### Вариант A: Автономная установка (Рекомендуется)
Теперь приложение можно установить как полноценную программу в меню камеры.
1. Скачайте `advanced_servo_drop_v2.0.0.zip` из релизов на GitHub.
2. В MaixVision перейдите в раздел **Device -> Install App**.
3. Выберите скачанный архив.
4. После установки приложение **"Servo Drop"** появится в главном меню на экране камеры.

#### Вариант B: MaixVision (для разработчиков)
1. Откройте `advanced_servo_app.py` в MaixVision.
2. Нажмите "Run" (F5) для быстрого теста.

### 4. Автозапуск при включении
Чтобы камера сама запускала сброс при подключении к повербанку на дроне:
1. Зайдите в **Settings** на экране камеры.
2. Выберите **Boot App**.
3. Выберите **Servo Drop**.

### 4. Первый запуск

#### Тест сервопривода:
```bash
# Сначала проверьте что серва работает
python3 servo_test.py
```

Серва должна двигаться через углы: 0° -> 45° -> 90° -> 135° -> 180° -> 90° -> 0°

#### Если серва не двигается:
- Проверьте подключение проводов
- Убедитесь что питание достаточное
- Попробуйте другой PWM пин (A19, A20, A21)

### 5. Запуск основного приложения

```bash
python3 advanced_servo_app.py
```

На экране появится:
- Статус сервы (OPEN/CLOSED)
- Статус готовности (ARMED/DISARMED)
- Режим детекции
- FPS счетчик

### 6. Базовое использование

#### Открыть меню:
- Коснитесь экрана в любом месте

#### В меню доступно:
1. **Detection Mode** - переключение между Color/Object/Motion
2. **Color Preset** - выбор цвета (Yellow/Red/Green/Blue/etc)
3. **Object Target** - выбор объекта (person/car/dog/cat/etc)
4. **Calibrate Color** - калибровка своего цвета
5. **ARM/DISARM** - включить/выключить готовность
6. **Save & Exit** - сохранить и вернуться

#### Калибровка цвета:
1. Откройте меню
2. Выберите "Calibrate Color"
3. Коснитесь объекта нужного цвета на экране
4. Готово! Теперь система будет реагировать на этот цвет

### 7. Примеры использования

#### Пример 1: Детекция желтого объекта
```
1. Запустите приложение
2. По умолчанию режим: Color Detection, Yellow
3. Покажите желтый объект камере
4. Серва откроется на 90°
5. Через 10 секунд закроется автоматически
```

#### Пример 2: Детекция человека
```
1. Откройте меню (tap на экране)
2. Выберите Detection Mode -> Object
3. Выберите Object Target -> person
4. Сохраните (Save & Exit)
5. Встаньте перед камерой
6. Серва откроется при обнаружении
```

#### Пример 3: Детекция движения
```
1. Откройте меню
2. Выберите Detection Mode -> Motion
3. Сохраните
4. Любое движение перед камерой вызовет срабатывание
```

### 8. Настройка конфигурации

#### Редактирование конфига на камере:
```bash
ssh root@10.144.30.1
vi /root/advanced_servo_config.json
```

#### Или скопируйте готовый пример:
```bash
# Смотрите config_examples.json для готовых конфигураций
# Например, для быстрого отклика:
{
  "close_delay": 3,
  "rearm_delay": 1,
  "detection_mode": "color",
  "color_preset": "Red"
}
```

### 9. Оптимизация производительности

#### Для максимальной скорости:
```json
{
  "camera_width": 320,
  "camera_height": 240,
  "detection_mode": "color",
  "min_area": 400
}
```

#### Для максимальной точности:
```json
{
  "camera_width": 1280,
  "camera_height": 720,
  "detection_mode": "object",
  "confidence_threshold": 0.7
}
```

### 10. Решение проблем

#### Серва дергается или не держит позицию:
- Недостаточное питание - используйте внешний БП 5V 2A
- Плохой контакт - проверьте провода

#### Низкий FPS (< 15):
- Уменьшите разрешение камеры
- Используйте Color Detection вместо Object Detection
- Увеличьте min_area

#### Ложные срабатывания:
- Увеличьте min_area (500 -> 1000)
- Для Object Detection увеличьте confidence_threshold (0.5 -> 0.7)
- Используйте калибровку для точной настройки цвета

#### Пропускает объекты:
- Уменьшите min_area (500 -> 300)
- Для Object Detection уменьшите confidence_threshold (0.5 -> 0.3)
- Улучшите освещение

1. **Всегда тестируйте серву отдельно** перед запуском основного приложения
2. **Используйте калибровку** для точной настройки цветов

4. **Сохраняйте конфиг** после каждой успешной настройки
5. **Экспериментируйте с порогами** для вашего конкретного случая
